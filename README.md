# Telegram-бот «Финсоветник»

## О проекте

Telegram-бот, демонстрирующий базовые и прикладные навыки разработки ботов на Python. Проект показывает, что исполнитель умеет создавать пользовательские сценарии внутри Telegram, подключать внешние API, сохранять данные в базу, организовывать пошаговый диалог с пользователем и оформлять интерфейс через кнопки.

Проект можно рассматривать как **демонстрацию практических навыков разработчика** для заказчиков, которым нужен Telegram-бот с меню, регистрацией, хранением данных и интеграцией со сторонними сервисами.

---

## Что показывает этот проект заказчику

Этот бот демонстрирует, что исполнитель умеет:

- разрабатывать Telegram-ботов на **Python + aiogram**;
- подключать и использовать **внешние API**;
- работать с **SQLite** и сохранять пользовательские данные;
- строить **пошаговые сценарии общения** с пользователем;
- реализовывать **кнопочное меню** и удобную навигацию;
- организовывать логику команд, обработчиков и состояний;
- создавать основу для дальнейшего масштабирования проекта.

## Реализованный функционал

### 1. Стартовое меню
После запуска бот приветствует пользователя и показывает меню с основными разделами.

В проекте реализованы кнопки:
- **Регистрация в боте**
- **Курс валют**
- **Советы по экономии**
- **Личные финансы**

### 2. Регистрация пользователя
Бот умеет регистрировать пользователя и сохранять его данные в базу данных.

Сохраняются:
- Telegram ID
- имя пользователя

Это показывает навык работы с пользовательскими данными и созданием простой БД для Telegram-бота.

### 3. Получение курса валют через API
Бот обращается к внешнему API и показывает пользователю актуальные значения валют.

Это демонстрирует:
- подключение стороннего сервиса;
- обработку ответов API;
- вывод данных в понятном формате внутри Telegram.

### 4. Блок с советами по экономии
Пользователь может получить случайный совет по экономии.

Этот модуль показывает, как в боте можно реализовывать полезный контентный блок и простую логику выдачи рекомендаций.

### 5. Сценарий «Личные финансы»
В боте реализован пошаговый диалог, в котором пользователь вводит категории расходов и суммы.

Сценарий включает:
- ввод первой категории и суммы;
- ввод второй категории и суммы;
- ввод третьей категории и суммы;
- сохранение введённых данных в базу.

Это демонстрирует важный для заказчиков навык — умение строить **FSM/сценарии состояний**, когда бот ведёт пользователя по шагам и собирает данные последовательно.

## Технологии и инструменты

В проекте используются:

- **Python**
- **aiogram** — для разработки Telegram-бота;
- **SQLite** — для хранения данных пользователей;
- **requests** — для запросов к внешнему API;
- **FSM (Finite State Machine)** — для пошагового сценария ввода финансовых данных.

## Какие навыки особенно хорошо видны в проекте

С точки зрения потенциального заказчика проект показывает следующие компетенции исполнителя:

### Разработка логики Telegram-бота
Исполнитель умеет создавать команды, обработчики сообщений, меню и сценарии взаимодействия с пользователем.

Исполнитель умеет подключать внешний источник данных и использовать его внутри Telegram-бота.

Бот не только отвечает на команды, но и ведёт пользователя по последовательному сценарию заполнения данных.
